# David Fernández-Cuenca Portfolio

A modern, professional portfolio for David Fernández-Cuenca Marcos, Software Engineer specialized in air traffic control systems. This project is structured for maintainability, scalability, and ease of collaboration.

---

## 📁 Project Structure

```
Portfolio_David/
│
├── public/                # Static assets and main HTML
│   ├── index.html         # Main portfolio page
│   ├── site.webmanifest   # PWA manifest
│   └── assets/            # (Recommended) Images, icons, and other static files
│
├── server.js              # Node.js static file server
├── README.md              # Project documentation
├── LICENSE                # License file
└── .gitattributes         # Git settings
```

> **Note:** If you add images, icons, or other static files, place them in `public/assets/` for better organization.

## 🚀 Overview

This portfolio showcases the skills, experience, and projects of David Fernández-Cuenca Marcos. It is built with modern web standards, optimized for SEO, accessibility, and responsive design. The project is ready for deployment and easy to maintain or extend.

## 🛠️ Setup & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/illuminatus503/Portfolio_David.git
   cd Portfolio_David
   ```

2. **Install dependencies:**
   (No dependencies required for the static site and server)

3. **Run the server:**
   ```bash
   node server.js
   ```

4. **Open in your browser:**
   [http://localhost:3000](http://localhost:3000)

---

## 🌐 Folder & File Descriptions

- **public/index.html**: Main HTML file. Contains all markup, styles (via Tailwind CDN), and scripts for interactivity, translations, analytics, and theme toggling.
- **public/site.webmanifest**: Manifest for PWA support and favicon configuration.
- **public/assets/**: (Recommended) Place for images, icons, and other static files.
- **server.js**: Minimal Node.js server for local development and static file serving.
- **README.md**: This documentation file.
- **LICENSE**: Project license (MIT).

## ✨ Features

- **SEO optimized**: Meta tags, Open Graph, Twitter Cards, canonical URLs.
- **Responsive design**: Works on desktop, tablet, and mobile.
- **Accessibility (A11y)**: ARIA labels, keyboard navigation, color contrast.
- **Dynamic content**: GitHub API integration for real-time project stats.
- **Contact form**: Validated, user-friendly, and ready for backend integration.
- **Theme toggle**: Light/dark mode with persistence.
- **Analytics**: Google Analytics integration and custom event tracking.
- **Modern UX**: Animations, scroll spy, interactive hero section.
