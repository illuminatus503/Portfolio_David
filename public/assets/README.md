# assets Directory

This directory is intended for all static assets used by the portfolio, such as images, icons, and other media files.

## Guidelines
- Store all images, icons, and static files here for better organization.
- Use descriptive filenames and subfolders if needed (e.g., `images/`, `icons/`).
- Reference assets in `index.html` or the manifest using relative paths (e.g., `/assets/your-image.png`).
- Keep this directory clean and remove unused files regularly.

**All documentation and filenames should be in English.**
