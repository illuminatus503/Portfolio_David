Favicon Test Page

This page tests all favicon options for David's portfolio.

Check the browser tab to see if the favicon appears.

Favicon Options:

  • Inline SVG (data URL) - Most reliable
  • favicon-simple.svg - Simple initials "DF"
  • favicon.svg - Robot design
  • favicon.ico - Traditional ICO format
  • PNG variants - Different sizes

Testing Instructions:

  1. Open this page in Chrome
  2. Check if favicon appears in the tab
  3. Try refreshing with Ctrl+F5
  4. Check browser developer tools for any 404 errors
